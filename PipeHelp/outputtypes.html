Script Output Types

In addition to scripts which output plain text, Pipe can also render image and HTML output. To change the output type of a script:

  1. Choose the type of output from “Script” > “Show Output As”.

In the script editor, this setting is saved with the script. In the script browser, it is only temporary and reverted back to the default as soon as you select a new script.

You can see the output type of a script by looking at its icon in the script browser:

  • The letter T represents text scripts.
  • Three small geometric shapes represent image scripts.
  • A small globe symbol represents HTML scripts.

Pipe uses built-in Mac OS X technologies for rendering images. As a rule of thumb, if the image or HTML output of your script can be displayed in QuickTime or Safari, it can also be displayed in Pipe.
