Using Pipe as a Service

Pipe provides a system-wide service which allows you to filter text from any application through scripts in your script browser, to create new scripts based on selected script code, and to use a text selection as input in your script browser.

To filter text through a script:

  1. Select text in any services-aware application (e.g. TextEdit, Safari or Mail).
  2. Choose “Services” > “Pipe” > “Filter Through Script…” from the application menu.
  3. In the window that pops up, select a script from the list. (This is the same list of scripts as in the script browser.)
  4. Change the “Reverse Transformation” setting as needed.
  5. If you would like to save the script output to a file, select “Save Output to File”.
  6. Click “OK”. (If you enabled the “Save Output to File” option, Pipe will ask you for a file name.) The results of the script should now appear in your application.

It is also possible to process files directly from the Finder:

  1. Select a single file in the Finder.
  2. Choose “Finder” > “Services” > “Pipe” > “Process with Script…”
  3. In the window that pops up, select a script from the list. (This is the same list of scripts as in the script browser.)
  4. Change the “Reverse Transformation” setting as needed.
  5. Click “OK”.

Note: This version of Pipe can only process one file at a time, but this may change in a future release.

You can also create new scripts based on a text selection, or use a selection as input in the script browser:

  1. Select the input or script text in any services-aware application.
  2. Choose “Services” > “Pipe” > “New Script from Selection” or “Use as Input” to create a new script or to copy the selection to the input box of the script browser, respectively.
