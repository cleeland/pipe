Passing keys or passwords to scripts

The recommended way is to use a variable of type “Password”. See Variables for more details about configuring script variables.

You can also pass the key as an argument, and check the argument list in your script. See Script Arguments for details.

Note that these methods are potentially unsafe if you share your computer with others: if someone else can run programs on your computer, they can intercept your key while the script is running. Future versions of Pipe may include other options for passing keys to your script. If this is important to you, please let us know!
