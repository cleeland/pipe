Line Endings

Each operating system uses different line ending characters: Mac OS X and UNIX-based systems use line feeds (LF), the classic Mac used carriage returns (CR), and Windows uses both characters (CR+LF).

Before Pipe executes your script, it converts the line ending characters both in the script source and it the input to the format configured in the preferences. By default, line endings in Pipe are LF. To change this setting, follow these steps:

  1. Choose “Pipe” > “Preferences…” to open the application preferences.
  2. Click “General” to open the general preferences.
  3. Select the line ending character(s) to be used from “Default Line Endings”.
