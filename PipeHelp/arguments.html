Setting Script Arguments

You can pass additional options and parameters to your script through script arguments.

  1. From either the script editor or the browser, choose “Script” > “Arguments…".
  2. Enter the arguments for your script, separated by white spaces.

To enter a single argument which contains one or more white spaces, enclose it in "double quotes".

You may use any of the Environment Variables provided by Pipe.

If your script is not executed directly, the arguments will be appended to the run command (see Run Commands).
