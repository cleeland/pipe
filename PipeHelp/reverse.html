Reverse Transformations

Sometimes it can be useful to switch between two modes of operations in a script: forward and reverse. Imagine a script which encrypts text. The normal mode of operation in this case is encryption, and the reverse mode is decryption.

  1. To toggle the transformation direction, choose “Script” > “Reverse Transformation”.

If Pipe is in reverse transformation mode, it will set the environment variable PIPE_REVERSE_TRANSFORM to 1 before it runs your script. In normal transformation mode, the variable is unset. Your script can check for this variable and change its mode of operation accordingly. If your script does not support reverse transformations, simply ignore the variable.

For information about other environment variables, see Environment Variables.
