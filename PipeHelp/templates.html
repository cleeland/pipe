Using Templates

Templates allow you to quickly create script with predefined settings and source code. To create a script based on a template:

  1. Choose a template from “File” > “New with Template”.
  2. Change the template as needed to create your script.
  3. Choose “File” > “Save” to save your script.

Follow these steps to create your own templates:

  1. Open a new script by choosing “File” > “New”.
  2. Enter the script code that should appear in the template.
  3. Change scripts settings as needed, e.g. by setting a run command.
  4. Choose “File” > “Save as Template…”.
  5. Enter a name for your template and click “Save”.

This saves the script in the correct folder for user templates (~/Library/Application Support/Pipe/Templates). You can also manually copy scripts into this folder or remove them to change the list of available templates.
