External Editors

Pipe allows you to edit scripts in your favorite editor. You can configure the editor which Pipe uses by following these instructions:

  1. Choose “Pipe” > “Preferences…” > “General” to open the general application preferences.
  2. Select your favorite editor from the list of external editors.

To edit a script or its input in the configured external editor:

  1. Click in the text box (either script source or input) which you would like to edit in the external editor.
  2. Choose “Edit” > “Edit in External Editor”. Pipe will open a window in your editor.
  3. You can leave the editor window open while you test and run your scripts.
  4. To edit the script in Pipe again, close the editor window.

Pipe will automatically pick up changes whenever you save the file in your editor. If auto-run is enabled, Pipe will run your script whenever you save, so you can see the results immediately.
