Using External Scripts

Pipe lets you easily create new scripts based on external plain-text script files:

  1. Choose “File” > “Open…”.
  2. Select the file that contains the script source code, and click “Open”.

Pipe will open a new script and fill in the script source from the file you chose. You can now enter some input and run your script. A Pipe script created in this way is separate from its original source file, but it keeps track of it. You can quickly update the original source file with your changes, and you can revert your script back to the version contained in the original source.

To update the source file with your changes:

  1. Hold down the Option key and choose “File” > “Save Script Source…”. Pipe will automatically select the source file you used to create this script.
  2. Click “Save” to update the file.

Tip: If you also hold down the Control key while choosing “Save Script Source”, Pipe will not ask for a file name and immediately update the original source file.

To revert your script code to its original source file revision:

  1. Hold down the Option key and choose “File” > “Revert to External Script Source…”.
