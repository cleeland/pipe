Auto-Run

Pipe can automatically run your scripts when you change important settings, so you can see the results immediately. To enable or disable the auto-run feature:

  1. Choose “Pipe” > “Preferences…” > “Running”.
  2. Enable or disable the option “Run scripts automatically”.

You can temporarily toggle this option by holding the Option key, for example while you switch between scripts in the Script Browser.

Pipe automatically runs scripts in several cases:

  • When you save a script in an external editor.
  • When you change script settings via “Script” > “Settings…”.
  • When you enter or modify script arguments.
  • When you reverse the transformation direction.
  • When you copy script output to input.
