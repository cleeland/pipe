Variables

Variables allow you to change the behavior of your scripts. For example, an encryption script could have a variable for the encryption password and another one for the encryption method (algorithm). All variables are passed as simple environment variables to your script. To add a new variable, follow these instructions:

  1. In your script editor, choose “Edit” > “Variables” to show the script's info drawer.
  2. In the drawer, choose “Add Variable” from the menu that appears when you click the corner button (down-pointing arrow) of the variable list.
  3. A new variable with the name “untitled” appears in the variable list. Double-click its name to change it to something more suitable.
  4. To edit the variable’s value, double-click in the “Value” column or press Return.

The new variable will be accessible from inside your script through an environment variable of the same name. For example, if your variable is named “password”, you can access it in a Bash shell script by typing “$password”.

Pipe supports four different variable types:

  • Text: A simple text variable. Its value is passed directly to your script without modification.
  • Password: This is treated like a text variable, but its contents are not displayed in Pipe, which makes it useful for passing passwords to your scripts.
  • Boolean: A variable that can be switched between only two values: on and off. If a boolean variable is on, the corresponding environment variable in your script will have the value “1”. If the variable is off, the environment variable will be unset.
  • Enumeration: A variable with a limited list of predefined values. You can configure all possible values by choosing “Enumeration Values…” from the corner menu. In the sheet that appears, enter all values separated by a comma (and no white spaces inbetween).

The values of variables can be changed in the script editor, in the script browser, and when invoking Pipe from the Services menu. However, the names, numbers, and types of variables can only be changed in the script editor.
