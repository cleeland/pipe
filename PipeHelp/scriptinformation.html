Entering information about scripts

You can set a home page for your scripts, so others can find out more about them and perhaps download the latest versions. You can also embed usage information which can be quickly viewed in the script browser.

  1. In the script editor, choose “Script” > “Settings…” > “Information”.
  2. Enter a home page URL for your script. You can also use your e-mail address, by prefixing it with mailto:
  3. Enter some basic usage instructions, e.g. what kind of input your script expects. You can use multiple fonts and colors and even drag small images into the text field.

To view this information in the script browser:

  • Choose “Script” > “Open Home Page” to open the script’s home page.
  • Choose “Script” > “Show Usage Information” to display the script’s usage instructions in a drawer.
