Script Support Files

You can include support files with your scripts, which will be distributed along with your script code:

  1. In the script editor, choose “Script” > “Support Files…”.
  2. In the sheet that appears, either click the “+” button to open a file selection dialog, or drag files from the Finder into the white area.

Support files can be accessed from your scripts through the script support directory specified in the PIPE_SCRIPT_SUPPORT_PATH environment variable. For example, if you have added a support file “dictionary.txt”, you can access it from your script through the path “$PIPE_SCRIPT_SUPPORT_PATH/dictionary.txt”.

Pipe keeps track of the files that were added to a script, and will automatically update them whenever the original file changes. In the example above, you could edit the “dictionary.txt” file in its original location, and Pipe would automatically update the script whenever it is run.

Important: Do not edit files stored inside a Pipe script bundle while the script is open in Pipe. Your changes will be overwritten when Pipe saves the script.
