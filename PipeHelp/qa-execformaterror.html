My script won’t run – “Exec format error”

“Exec format error” means that Mac OS X did not recognize the type of your script.

  1. Make sure your script starts with the two characters #! followed by the path to your script interpreter.

Some script cannot be executed directly by Mac OS X. You can usually execute these with a Run Command.

You may also use one of the predefined templates to create a new script, or to see how a correct script should look like:

  1. Choose the kind of script you would like to create from the menu “File” > “New with Template”.

For information about templates, see Using Templates.
