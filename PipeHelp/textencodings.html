Text Encodings

By default, Pipe uses and expects the UTF-8 text encoding for script source, input and output. To change the encoding options for a script:

  1. In the script editor, choose “Script” > “Settings”.
  2. Click the “Text Encoding” tab.
  3. Choose the text encodings for script source, input and output.

You can also change the default text encoding for new scripts:

  1. Choose “Pipe” > “Preferences…” > “General”.
  2. Select the desired default text encoding.
