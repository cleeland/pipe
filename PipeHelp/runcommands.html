Run Commands

Sometimes you don’t want your script to be executed directly, but instead to be processed by another program.

  1. In the script editor, choose “Script” > “Settings…”.
  2. Switch to the “Running” tab.
  3. Select “Run script with the following command”.
  4. In the text box below, enter the full path to the program followed by any required arguments.

The string $script in the entered command will be replaced with the path to your script source. You can also use any of the environment variables provided by Pipe.

If any part of the script command or its arguments contains white spaces but should be treated as a single entity (e.g. file names), remember to enclose it in "double quotes".
